== Description ==

Light plugin for save all search engine keywords and publish it in widget. If you want help or donate, please visit to <a href="http://s-one.ru/news/wordpress-plagin-istorija-poiska.html">official web page of plugin</a>.

== Installation ==

Follow by steps:

1. Upload `/se-referer/` directory to the `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add widget on disign -> widgets menu
